# YLChart
YLChart  主要是用来绘制环状图表
===
包含内容:<br> 
>>>>>* 1 渐变色圆环进度条<br>
>>>>>* 2 绘制多色环状图.按顺序绘制。从0 ~ 1.<br>
>>>>>* 3 绘制多色环状图.多段同时绘制<br>
>>>>>* 4 渐变色弧形绘制<br>
>>>>>* 5 折线图，填充色渐变色填充<br>
>>>>>* 6 绘制间隔弧度的多色圆环<br>
